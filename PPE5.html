Déploiement serveur Wordpress

Le serveur web Nginx, combiné avec MySQL et PHP, forme une solution robuste pour héberger des sites WordPress.

Pour commencer, l'installation de Nginx permet de servir les fichiers web. Ensuite, l'installation de MySQL est essentielle pour gérer la base de données de WordPress. La dernière composante est PHP, un langage de script côté serveur. WordPress est écrit en PHP, ce qui rend ce langage crucial pour le fonctionnement du site. Une fois ces composants installés et configurés, vous pourrez installer WordPress et commencer à créer votre site sur une plateforme puissante et flexible.

1. Objectifs

L'objectif de ce projet est de mettre en place un serveur web efficace et performant pour héberger des sites WordPress.

2. Déroulement

Étape 1 : Prérequis

  • Un serveur Linux
  • Accès en tant que superutilisateur su -
  • SFTP: On peut copier des fichier d'autre machine avec sftp sftp user0@172.16.246.10 tant que ssh est installer sur la machine a distance

Étape 2 : Installer et configurer Nginx

  • 2.1. Mettez à jour les paquets du système : apt update
  • 2.2 Installez Nginx : apt install nginx
  • 2.3 Démarrez le service Nginx : systemctl start nginx
  • 2.4. Vérifiez si Nginx est en cours d'exécution : systemctl status nginx
  • 2.5 Créez un fichier de configuration pour WordPress : nano /etc/nginx/sites-enabled/EMXPnw52
  • 2.6 Collez le contenu suivant dans le fichier :
            server { 
              listen 80;
              listen [::]:80;
              
              server_name EMXPnw52.eval244.yajako.info.lan;
              
              root /var/www/EMXPnw52;
              index index.php;
                location ~ .php$ {
                  include snippets/fastcgi-php.conf;
                  fastcgi_pass unix:/run/php/php7.4-fpm.sock;
                }   
             }
  • 2.7 Si la syntaxe est valide, redémarrez Nginx : systemctl restart nginx

Étape 3 : Installer MySQL et PHP

  • 3.1. Installez le serveur MySQL : apt install mariadb-server
  • 3.2 Installez PHP et les extensions requises : apt install php-fpm php-mysql

Étape 4 : Télécharger et installer WordPress

  • 4.1 Accédez au répertoire /var/www/ : cd /var/www/
  • 4.2 Téléchargez la dernière version de WordPress depuis le site officiel (https://wordpress.org/) : wget https://wordpress.org/latest.zip
  • 3. Extrayez l'archive téléchargée : unzip latest.zip
  • 4. Renommez le répertoire extrait en "EMXPnw52" : mv wordpress/ EMXPnw52
  • Définissez les permissions appropriées sur le répertoire WordPress : 
    chown -R www-data:www-data /var/www/EMXPnw52  
chmod -R 755 /var/www/EMXPnw52

Étape 5: Créer une base de données MySQL pour WordPress

  • 5.1. Connectez-vous à MySQL en utilisant la commande suivante : mysql -u root
  • 5.2. Une fois connecté, créez une nouvelle base de données pour WordPress : CREATE DATABASE EMXPnw52;
  • On peut la voir en tapant show databases;
  • 5.3 Créez un nouvel utilisateur et attribuez-lui un mot de passe : 
    CREATE USER 'EMXPnw52'@'localhost' IDENTIFIED BY 'EMXPnw52';
  • 5.4 Accordez à l'utilisateur des privilèges sur la base de données : 
    GRANT ALL PRIVILEGES ON EMXPnw52.* TO 'EMXPnw52'@'localhost';
  • 5.5 Mettez à jour les privilèges : 
    FLUSH PRIVILEGES;

Étape 6 : Configuration Base de donnée manuellement (facultatif)

La Base de donnée doit ce connecter automatiquement quand on lance l'interface graphique de Wordpress mais certaine fois cela ne fonctione pas.

  • 6.1 Renommez le fichier de configuration WordPress :
    mv /var/www/EMXPnw52/wp-config-sample.php /var/www/EMXPnw52/wp-config.php
  • 6.2. Ouvrez le fichier wp-config.php: 
    nano /var/www/EMXPnw52/wp-config.php
  • 6.3 Modifiez les lignes suivantes en fonction de vos paramètres de base de données MySQL : 
    define('DB_NAME', 'EMXPnw52');
define('DB_USER', 'EMXPnw52');
define('DB_PASSWORD', 'EMXPnw52');
define('DB_HOST', 'localhost');
define('DB_CHARSET', 'utf8');
define('DB_COLLATE', '');
define( 'WP_DEBUG', true );
  • 6.4. Enregistrez le fichier et quittez l'éditeur.

Étape 7 : Accéder à WordPress

  • 7.1 Ouvrez votre navigateur Web et accédez à l'URL suivante : EMXPnw52.eval244.yajako.info.lan
  • 7.2 Suivez les instructions à l'écran pour configurer votre site WordPress, créer un compte administrateur, etc
